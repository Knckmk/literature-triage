# Path TSP: from 5/3 to 3/2
We trace the line of work on path TSP, which asks for a Hamiltonian path of minimum length in a metric space.
Improvements from 5/3 toward 3/2 have leveraged matroid intersection, randomized rounding, and parity correction.
The path variant is a strict generalization of metric TSP, so any path-TSP technique gives metric-TSP intuition.
We discuss connections to ear decompositions and the Held-Karp LP.
